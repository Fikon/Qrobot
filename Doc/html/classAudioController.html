int AudioController::audioReadIn ( )

录音

从设备中读取音频数据，将数据储存在类成员buffer中

调用该函数的方法应该是在一个循环当中，每次调用之后应该立即 使用buffer中读取的音频数据，防止被下一次输入所覆盖

Return values:
-1error
0success
1overrun 表示处理buffer缓存区不够快而被新的数据覆盖了
2shortread 表示buffer缓存区未读满
